# Voice Sample

我做这个项目的时候比较在意两件事。第一，项目不能只停留在“我能改写一段话”，因为这样很难说明方法是否可复现。第二，命令必须能直接跑，不然换一台电脑或者给别人演示的时候就会很麻烦。

所以我更倾向于把流程拆开：先检测原文，再生成改写提示，最后把修改前后的分数写进报告。这样就算没有外部 API，也能用本地规则跑出一个可复现的 AI-like Rate。这个分数不等于真正的检测器结论，但可以作为项目里的实验指标。

如果后面要继续改，我会优先补两块：一块是接入真实 API 的示例结果，另一块是把更多中文 AI 套话整理成 pattern catalog。这样项目会更像一个可以继续扩展的 skill。
